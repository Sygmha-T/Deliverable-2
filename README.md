This is a python program that works like a smart calculator for student grades.
The program accepts inputs for test scores, assignment and exam.
It then perform arithmethic operations to calculate the total score of the student.
It assigns a corresponding grade for the score calculated and identifies who passes, fails, or did exceptionally well.
It allows multiple calculations for different subject or courses and can add all scores together.
At the end of each session, it prints out the total score and percentage score of all the sessions all together.
